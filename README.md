# Mother Margaret Junior School Backend

Complete backend system for Mother Margaret Junior School and Daycare in Namuyenje, Mukono, Uganda.

## Features
- ✅ Admin Portal with 7 initial admin accounts
- ✅ Teacher Portal with Zoom integration
- ✅ Parent Portal with mobile money payments
- ✅ Student Portal with educational games
- ✅ AI Assistant (DeepSeek integration)
- ✅ Website Management System
- ✅ Real-time notifications
- ✅ Database with PostgreSQL

## API Endpoints
- `GET /health` - Health check
- `GET /api/school/info` - School information
- `GET /api/init` - Initialize system
- `POST /api/auth/login` - User login
- `GET /api-docs` - API documentation

## Deployment on Render.com
1. Create new Web Service
2. Connect GitHub repository
3. Set start command: `npm start`
4. Add PostgreSQL database
5. Set environment variables
